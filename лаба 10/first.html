FLEX-BOX
Flexbox — это мощный инструмент CSS, который предназначен для создания гибких и адаптивных макетов. Основная идея Flexbox — управлять расположением элементов вдоль одной из осей: горизонтальной (main axis) или вертикальной (cross axis). Это особенно полезно, если нужно выровнять элементы, равномерно распределить их или создать динамическую структуру. Для использования Flexbox у родительского элемента нужно задать свойство display: flex;. После этого все его дочерние элементы автоматически становятся флекс-элементами. Система работает по принципу контейнера и элементов внутри него. Родительский элемент определяет поведение флекс-элементов с помощью специальных свойств. Одно из ключевых свойств Flexbox — это flex-direction, которое задаёт направление главной оси. Например, можно располагать элементы горизонтально с помощью значения row (по умолчанию) или вертикально с column. Существуют и обратные направления — row-reverse и column-reverse, которые меняют порядок элементов. Для управления выравниванием используется свойство justify-content. Оно определяет, как элементы располагаются вдоль главной оси. Например, с помощью значений flex-start, flex-end, center, space-between или space-around можно задать разные типы распределения элементов, такие как выравнивание по краям, центрирование или добавление отступов между ними. По дополнительной оси элементы управляются через свойство align-items. Оно помогает выровнять элементы внутри контейнера вертикально, если главная ось — горизонтальная. Значения flex-start, flex-end, center, stretch или baseline задают, как именно элементы располагаются относительно контейнера. Чтобы управлять выравниванием всего содержимого внутри контейнера, используется свойство align-content. Оно работает, если флекс-элементы распределены на нескольких строках (например, когда используется flex-wrap: wrap). Это свойство помогает задать отступы между строками или выровнять их относительно контейнера. Флекс-элементы можно настраивать индивидуально с помощью свойства align-self. Оно переопределяет align-items для конкретного элемента, позволяя выровнять его независимо от остальных. Ещё одно полезное свойство — order, которое задаёт порядок отображения элементов в контейнере, даже если они идут в другом порядке в HTML. Свойство flex — это универсальный инструмент для управления размерами элементов. Оно объединяет три составляющих: flex-grow (растяжение), flex-shrink (сжатие) и flex-basis (базовый размер). Это позволяет гибко задавать, как элементы будут изменять свои размеры в зависимости от доступного пространства. Flexbox делает работу с макетами простой и удобной, особенно когда нужно адаптировать интерфейс под разные размеры экранов. С его помощью можно легко создать как простые центровки, так и сложные структуры, не прибегая к хаотичным отступам и позиционированию.
